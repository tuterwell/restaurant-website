# restaurant-website   
本網站含有餐廳清單，可點選餐廳圖片了解詳細資訊。另外亦具備搜尋功能，可快速找到目標餐廳。  

## 如何使用  
1. 請先下載所有檔案
   `git clone https://github.com/tuterwell/restaurant-website`
2. 確保有安裝 node.js
3. 安裝 npm
   `npm install`
4. 開始執行
   `npm run dev `
   跑出 ` Server is listening on port 3000` 即算成功
5. 若要結束請按 `ctrl + c`

## 開發工具  
* Node.js 18.16.1
* Express 4.17.1
* Express-Handlebars 5.3.3
* Bootstrap 5.2.1
